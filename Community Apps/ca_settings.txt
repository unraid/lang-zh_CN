Hide Deprecated Applications=隐藏已弃用的应用程序
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=此设置隐藏可用应用列表中已弃用的应用程序，但已弃用的应用仍将显示在已安装的应用和以前的应用中
Hide Incompatible Applications=隐藏不兼容的应用程序
This setting will hide applications that are listed as being incompatible with your version of Unraid=此设置将隐藏列出的与您的 Unraid 版本不兼容的应用程序
Enable Reinstall Default=启用默认重新安装
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=此选项启用卡片和弹出窗口上的另一个按钮，该按钮允许您使用默认模板值重新安装已安装的应用程序。此选项有时会令人混淆，通常在您希望安装并且同时运行应用程序两次时才使用
Enable additional search results from dockerHub=启用 dockerHub 中的其他搜索结果
This setting enables CA to retrieve additional search results from dockerHub Note that docker must be enabled for this to be operational=此设置支持 CA 从 dockerHub 检索其他搜索结果。请注意，必须启用 docker 才能使其正常运行
Allow CA to send any emergency notifications=允许 CA 发送任何紧急通知
This setting allows CA to periodically scan your system and alert you to the presence of=此设置支持 CA 定期扫描系统并在存在此项目时向您发出警报
